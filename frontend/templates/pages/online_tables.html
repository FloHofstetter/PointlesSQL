Online Tables

An online table is a low-latency copy of a Delta table, synced into a Postgres or SQLite serving database for point lookups by primary key. Full mode replaces the target on every sync; CDF mode applies Change Data Feed events incrementally from the last synced version. Target URLs may reference credentials as {{ '{{secrets//}}' }} — the placeholder is stored, never the secret.
