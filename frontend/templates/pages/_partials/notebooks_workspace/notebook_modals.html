{# Notebook create/rename/delete modals. Driven by the ``notebookDialogs()`` mixin spread into ``notebookWorkspace()``. Two modals live here: - ``pql-notebook-path-modal`` — used by both create and rename (mode-aware title + submit label). - ``pql-notebook-delete-modal`` — single confirm dialog. Toggle pattern: ``:class="{ 'show d-block': flag }"`` + a backdrop inline style. Bootstrap's JS modal API is intentionally NOT used here — Alpine 3.14 strips ``display: block`` set externally when ``x-show`` toggles, which is why we drive visibility via class binding instead (see ``feedback_bootstrap_modal_x_show``). #}
