{# =================================================================== Hidden / discoverable features =================================================================== #}

Hidden features

Things that aren't on a button label but live one click away. Find them once, and they stick.

Collapse the primary rail & side panel
Top-left of the topbar — two chevron buttons. The first collapses the icon rail from 220 px to 64 px; the second hides the contextual side panel entirely so the main content gets its full width. Both states persist across reloads.
Feed density modes
On /feed — three-button group on the filter bar (Comfortable / Compact / Headlines). Headlines collapses each item to a single line; useful for scanning a long overnight queue. Choice persists.
Feed item ⋯-menu
Each row carries a dot menu with Mark read, Mute thread, Mute author, Snooze 1 h / 8 h / 1 d and Copy link. Mutes filter the row out of every future feed load until you un-mute from the settings page.
Saved searches
On /feed → Discover tab → Saved searches. Click Save current to bookmark the Activity tab's current filter + kind + search combo; click a saved row to apply it. Stored per-browser in localStorage.
Live feed updates
/feed opens a Server-Sent-Events stream and queues new activity rows under an X new banner — click the banner to drain. The small dot at the top-right of the filter bar shows the connection state.
Branch preview & promote
Every Delta-branch row (under /branches) carries a Preview action that diffs the branch against its parent and a Promote action that pointer-swaps after the supervisor approval review passes.
Time-travel any table
On any table detail page, the History tab lists every Delta version with timestamp + operation; admins also see a Row at version form for ad-hoc forensic queries.

{# =================================================================== Glossary =================================================================== #}

Glossary

Catalog · Schema · Table
The Unity-Catalog three-level namespace. Every PQL primitive that reads or writes a Delta table addresses it as catalog.schema.table.
Data product (DP)
A curated, contract-bearing dataset registered in PointlesSQL. Carries owner, SLA, schema-change policy, README, discussion, reviews and citations. Lives at /data-products.
Model
An MLflow model registered as a Unity-Catalog MODEL securable. Audit-linked to the agent run that trained it; lives at /models.
Branch
A zero-copy Delta-Lake shallow clone of a schema, scoped to one agent run. Lets the run write freely against isolated tables; an explicit Promote step pointer-swaps the parent.
Run
One supervised invocation of an agent. Every PQL primitive it calls is captured as a row-level + column-level lineage operation with cost estimate and source-byte hash.
Notebook · Saved query
JupyterLab notebooks and ad-hoc SQL editor snippets, both audited and citation-linkable from any comment, review or issue.
