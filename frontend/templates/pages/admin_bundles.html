Asset bundles

Declare jobs, pipelines, and BI dashboards as one YAML bundle, then plan + apply it idempotently. Plan diffs each resource by its identity (job name / pipeline slug / dashboard slug) and never deletes top-level resources — unmanaged ones are listed as orphans. Widgets of a managed dashboard belong entirely to the bundle: undeclared widgets are removed on apply.
