{# contextual side-panel (240 px). — six static branches replaced by per-section ``components/sidebars/*_sidebar.html`` partials with their own Alpine factories (see ``frontend/js/components/sidebars/``). Sits to the right of the icon-rail at >= md. The contents are a function of ``active_section`` (computed in base.html from ``active_page``): * federation → catalog tree (components/sidebar.html) * dashboards → dashboards tree (components/dashboards_sidebar.html) * runs → recent agent runs grouped by status * branches → active/promoted/discarded delta-branches * workspace → flat list of every notebook * jobs → active/paused jobs * alerts → enabled/disabled alerts * mlflow → recent UC-registered models * sql / admin → small static link lists (kept inline below) Federation is the catch-all default for sections without a dedicated branch (volumes, models — both already render inside the catalog tree per.x). On mobile, ``base.html`` mounts this same partial inside the bootstrap offcanvas drawer; ``components/context_panel.css`` drops the sticky/border chrome there. #}
